    z

    @function z($name, $z-indexes) { ... }

    Description

    Z-Index Manager

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    String provided at instantiation, referring to the element having Z-Index set.

    		String— none
    $z-indexes

    List of string names of elements; ranked in descending order of Z-Index.

    		Variable— none

    Example

    .component {
  z-index: z('Modal')
}

    Requires

    • [variable] z-indexes — List of string name representations of elements to be ordered.
